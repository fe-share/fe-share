less cnpm i css-loader style-loader less-loader less { test: /\.less$/, use: [ {loader:'style-loader'}, {loader:'css-loader'}, {loader:'less-loader'} ] }, 注意顺序
sass cnpm i css-loader style-loader sass-loader node-sass { test: /\.scss$/, use: [ {loader:'style-loader'}, {loader:'css-loader'}, {loader:'sass-loader'} ] } 注意顺序 注意的就是sass-load而容易报错,非常非常容易
配置方法都一样 loader的写法有几种, 1.命令行的形式, webpack --module-bind 'css=style!css' 2.写在引入的文件里面 use: 'url-loader?limit=1024&name=[path][name].[ext]&publicPath=output/' import './style.css!style-loader!css-loader' 3.use: [ 'style-loader', 'css-loader' ] 4. use: [ {loader:'style-loader'}, {loader:'css-loader'}, {loader:'sass-loader'} ], 推荐使用第四种,因为第四种可以设置配置项
